# CurrencyXchange

A simple web application that displays live currency exchange rates using the Frankfurter API.

## Purpose

This project allows users to view current exchange rates for different currencies, with the ability to select a base currency.

## API Used

- Frankfurter API: https://api.frankfurter.app/latest

## Features

- Select base currency (USD, EUR, GBP, INR)
- Fetch and display live exchange rates
- Responsive design for mobile, tablet, and desktop

## Technologies

- HTML
- CSS
- JavaScript (Fetch API)

## Setup and Run

1. Clone the repository
2. Open index.html in a web browser
3. Or run a local server: `python3 -m http.server 8000` and open http://localhost:8000
